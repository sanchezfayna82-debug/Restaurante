# Sistema de Pedidos de Restaurante

## Descripción
Programa Java para la UT4 - Actividad 4. Gestiona pedidos de restaurante con validaciones.

## Cómo ejecutar en VS Code
1. Abre la carpeta RESTAURANTE en VS Code
2. Abre `src/Main.java`
3. Haz clic en "Run" o presiona F5

## Archivos
- `src/Articulo.java` - Clase para artículos
- `src/EstadoPedido.java` - Estados del pedido
- `src/Pedido.java` - Clase principal del pedido
- `src/Main.java` - Programa principal

## Características
- ID automático para cada pedido
- Máximo 5 artículos por pedido
- No se puede retroceder de estado
- Cálculo automático del total
