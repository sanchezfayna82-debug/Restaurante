# Restaurante –
 Gestión de Pedidos

## Descripción – 
Programa en Java que representa un **pedido de restaurante** utilizando POO, encapsulación, enum y atributos/métodos estáticos. Permite crear pedidos, agregar hasta 5 artículos, calcular total, cambiar estado del pedido y mostrar resumen.
